# MoneyWall


# Getting Started
Clone MoneyWall repo using git:
```
git clone https://github.com/agamy5280/Sprints-Grad-Project.git
```
# Front-End
Install the dependencies:
```
npm install
```
Run Application
```
ng serve
```
# Back-End
```
cp .env.example .env
```
Install the dependencies:
```
npm intall
```
composer install
```
composer update
```
php artisan key:generate
```
php artisan storage:link
```
php artisan migrate
```
php artisan serve
```
